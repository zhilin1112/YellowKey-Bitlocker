## Overview
**YellowKey-Bitlocker** is a lightweight and user-friendly Windows utility designed to help users manage BitLocker encrypted drives. It simplifies the process of unlocking, recovering, and working with BitLocker-protected volumes.

> [!Important]
> Always backup your recovery keys before using any BitLocker management tools.

## Key Features
- Easy unlocking of BitLocker drives
- Recovery key management
- Simple and clean interface
- Support for fixed and removable drives
- Password and recovery key authentication
- Portable version (no installation required)

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `YellowKey-Bitlocker.exe` as Administrator
3. Select the encrypted drive and unlock it with password or recovery key

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| Drive not detected         | Make sure the drive is connected properly     |
| Unlock fails               | Double-check your password or recovery key    |
| Antivirus warning          | Add the folder to exclusions                  |
| Permission error           | Run the program with administrator rights     |
